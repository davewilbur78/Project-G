# Module 04: Citation Builder

## Description

The Citation Builder generates fully formed citations in Evidence Explained style for any source type encountered in genealogical research — vital records, census records, land records, probate records, newspapers, photographs, databases, websites, and more. For each citation the module collects the required fields through a structured interview, classifies the source as Original, Derivative, or Authored, identifies whether the information is Primary or Secondary (based on the informant's knowledge), and generates both a full citation and a short reference note. The Image Citation Builder v2 prompt engine powers citation generation for uploaded document images.
