# Module 04: Citation Builder

## Description

The Citation Builder generates fully formed citations in Evidence Explained style for any source type encountered in genealogical research — vital records, census records, land records, probate records, newspapers, photographs, databases, websites, and more. For each citation the module collects the required fields through a structured interview, classifies the source as Original, Derivative, or Authored, identifies whether the information is Primary or Secondary (based on the informant's knowledge), and generates both a full citation and a short reference note. The Image Citation Builder v2 prompt engine powers citation generation for uploaded document images.

## Key Inputs

- Source metadata collected through structured interview (repository, title, author, date, URL, page, etc.)
- Source type (vital record, census, land record, probate, newspaper, photograph, database, website, etc.)
- Uploaded document image (optional — triggers image citation path)

## Key Outputs

- Full Evidence Explained-style citation
- Short reference note form
- Source classification: Original, Derivative, or Authored
- Information classification: Primary or Secondary (based on informant's knowledge)

## GPS Touchpoints

- Directly enforces complete and accurate citations (GPS element 2)
- Applies GPS source and information classification terminology throughout
- Prevents the use of non-GPS terms ("primary source," "secondary source") in all output

## Prompt Engines Used

- **Image Citation Builder v2** — generates citations for uploaded document images
- **GRA v8.5c** — GPS enforcement layer applied across all output

## Data Written to Supabase

- `sources` — source records with type classification and full citation text
- `citations` — citation records linking specific facts to sources, with information classification

## Connection to Other Modules

- Citations are consumed by every other module in the platform
- Source classifications feed Document Analysis Worksheet (05) and Source Conflict Resolver (06)
- Citations appear in Timeline Builder (07), Family Group Sheet Builder (11), Research Report Writer (09), and Case Study Builder (10)
- File names for uploaded documents are generated by File Naming System (13) using citation metadata
