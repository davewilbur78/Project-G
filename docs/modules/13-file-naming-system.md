# Module 13: File Naming System

## Description

The File Naming System generates standardized file names and folder structures for all research documents, images, and downloads. A consistent naming convention is essential for a research collection that spans decades and thousands of files. The module applies a configurable naming schema — typically encoding the subject surname, given name, record type, date, and repository in the filename — and generates the correct name for any document based on its citation metadata. The module also suggests the correct folder path within the researcher's local archive structure. File names generated here are stored in Supabase alongside the document record so files can always be located and re-cited.
